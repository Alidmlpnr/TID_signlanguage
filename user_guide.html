Türk İşaret Dili → Metin Dönüştürücü

Proje Hakkında

Bu uygulama, kamera üzerinden alınan görüntülerdeki Türk İşaret Dili (TİD) el hareketlerini analiz ederek gerçek zamanlı olarak metne dönüştürür.

Sistem, derin öğrenme modeli ve görüntü işleme teknikleri kullanılarak geliştirilmiştir.

Uygulama Nasıl Kullanılır?

  • Başlat butonuna tıklayın
  • Kamera erişimine izin verin
  • Elinizi kameraya gösterin
  • Sistem harfi otomatik olarak algılar
  • Algılanan harfler metin alanına eklenir
```
Harflerin doğru algılanması için eliniz net görünmelidir.
```

İşaret Algılama Sistemi

Sistem, el landmark (eklem noktaları) kullanarak işaretleri analiz eder. Hem tek el hem de iki el ile yapılan işaretler desteklenmektedir.

Metin Oluşturma

  • Harfler: Otomatik eklenir
  • space: Boşluk bırakır
  • delete: Son karakteri siler

Sistem Gereksinimleri

  • Kamera (webcam)
  • Modern web tarayıcı (Chrome önerilir)
  • İnternet bağlantısı

Dikkat Edilmesi Gerekenler

  • Işık ortamı yeterli olmalıdır
  • Arka plan sade olmalıdır
  • El kadraj dışına çıkmamalıdır
```
Düşük ışık veya karmaşık arka plan doğruluğu düşürebilir.
```

Teknik Bilgi

  • Model: MobileNetV3 (ONNX)
  • Görüntü işleme: MediaPipe Hands
  • Çalışma: Tarayıcı (client-side)
